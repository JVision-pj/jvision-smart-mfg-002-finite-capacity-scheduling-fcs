# 有限產能排程管理艙

以實際設備產能與工時限制為基礎的精細化排程工具，常為APS的細部執行層。

- 原始系統：有限產能排程系統（Finite Capacity Scheduling, FCS）
- 產業分類：智慧製造｜製造管理
- 使用部門：生管、製造工程、現場班組長
- 工作流程：產能建檔 → 訂單優先序設定 → 排程演算 → 甘特圖確認 → 現場派工
- AI 應用：AI學習歷史換線時間優化排程順序、智慧建議插單最佳插入點
- 主要 KPI：設備稼動率、換線次數、排程達成率
- 代表產品：Asprova、PlanetTogether、Preactor（Siemens Opcenter APS前身）
